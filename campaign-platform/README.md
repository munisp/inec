# INEC Campaign Intelligence Platform

A full-stack campaign management and election simulation platform built with React 19 + Tailwind 4 + Express 4 + tRPC 11.

## Features

- Digital Twin Simulation Engine with Monte Carlo analysis
- AI Scenario Narrative (LLM-powered briefings)
- Sensitivity Heatmap (Weather Severity x Security Threat)
- Run Comparison View with colour-coded KPI deltas
- Share Report (clipboard copy for WhatsApp/email)
- CSV/PDF Export for simulation history
- 22-feature Campaign Hub

## Stack

React 19, Tailwind CSS 4, shadcn/ui, Express 4, tRPC 11, Drizzle ORM, MySQL/TiDB, Manus OAuth
